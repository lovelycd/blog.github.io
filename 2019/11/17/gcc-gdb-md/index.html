gcc-gdb使用

程序生成可执行文件的过程：　　预处理，编译，汇编，连接

gcc

-E　　　　只进行预处理指令，生成文件格式与源文件相同

-S　　　　不进行汇编之后的操作，生成.s文件（汇编语言程序）

-c　　　　-c选项不进行连接处理，生成汇编文件.o（只编译，不链接形成的文件，里面包含了对各个函数的入口标记）

-o　　　　-o选项指定生成的文件名称

-g　　　　-g选项在编译时生成调试信息

gdb
